Keyboard Navigation

The Keyboard Navigation module permits traversel through a set of elements using the arrow keys.

Option Description Possible Values
Orientation Layout of the elements. Determines which keys are used to traverse in certain directions. Horizontal, Veritcal, Grid
Classes These classes will be added to the selected element. string
KeepClassOnBlur When set to true, the classes will remain on the last selected element after focus has moved. false (default), true
Width For use with grids. The width of a single row. int
